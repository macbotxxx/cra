Log in to MyCRA

You cannot apply for emergency support payments through this service. For more information, go to Canada Emergency Response Benefit or Canada Emergency Student Benefit.

First time logging in?

If you are new to the MyCRA app, you can do the following:

  • If you have an online account with one of our Sign-In Partners, log in using your existing sign-in information
  • Log in using your existing CRA user ID and password by selecting "CRA login"

If you do not have a CRA user ID, select “CRA register.”

Note

If you are registering through a Sign-In Partner or for a CRA user ID, you will need to have details from your most recent income tax and benefit return on hand.

What are your CRA user ID and password used for?

You can use your CRA user ID and password to access many services, including:

  • MyCRA – A web-based mobile app for individuals
  • MyBenefits CRA – A web-based mobile app for benefit recipients
  • My Account – A full service portal for individuals

These services can be accessed using any CRA user ID and password.
